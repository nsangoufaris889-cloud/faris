💻 What is a Computer?

A computer is an electronic machine that helps us work, learn, play games, draw pictures, and solve problems.

Examples:

  • Laptop
  • Desktop
  • Tablet

🖥 Hardware

Hardware means the parts of a computer that you can touch.

  • Monitor
  • Keyboard
  • Mouse
  • CPU
  • Printer

📱 Software

Software is the programs inside a computer.

  • Paint
  • Microsoft Word
  • Calculator
  • Games

⌨ Keyboard

The keyboard helps us type letters and numbers.

  • Enter Key
  • Space Bar
  • Shift Key
  • Backspace

🌍 Internet

The Internet lets people find information, watch videos, play games, and communicate with others.

Always use the Internet safely and ask an adult before sharing personal information.
